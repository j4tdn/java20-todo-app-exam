4. Phân biệt git merge và git rebase(giống, khác nhau, khi nào nên sử dụng)
Giống nhau: Được kết quả như nhau
Khác nhau: 
+ merge: 1 lần gộp
+ rebase: gộp nhiều lần

5. Phân biệt git fetch và git clone
git fetch: đã cập nhật lại version mới nhất 
git clone: copy lấy toàn bộ .git ở remote về local
